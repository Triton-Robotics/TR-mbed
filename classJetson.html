void Jetson::read ( )

Should be called once per loop. Reads the next set of data from the serial port.

// Example usage
Jetson jetson(PA_9, PA_10);
while (true) {
jetson.read();
printf("X: %f, Y: %f, Z: %f\n", jetson.get(Jetson::X), jetson.get(Jetson::Y), jetson.get(Jetson::Z));
}
