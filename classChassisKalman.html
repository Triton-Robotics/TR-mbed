Detailed Description

Kalman filter for Chassis localization.

States are:

  • x position (inches)
  • x velocity (inches / second)
  • y position (inches)
  • y velocity (inches / second)
  • heading (degrees)
  • rotation (degrees / second)

Measurements are:

  • LF velocity (inches / second) ... veloctities for RF, LB, and RB
  • IMU angle (degrees)
